### 说明

ts-cnode的SSR改造版本，原始项目的git地址为：https://github.com/xhonker/ts-cnode

#### 技术栈

> Vue, Vue Router, Vuex, typescript, scss, webpack, Vue SSR

### 初始化
```
npm install
```

### 编译
```
npm run build
```

### 启动服务
```
npm run start
```

### Lints and fixes files
```
npm run lint
```
